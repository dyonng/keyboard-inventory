# Quick reference guide for IKBC F87/F108 Keyboard

_Since the manual for this keyboard is hard to find and difficult to understand, I wrote a quick reference guide on how to set the RGB lights. It's a great keyboard, I've have the all aluminum MF108 in black for over a year but I never change the lights colors since I never remember the shortcuts... Hopefully this quick reference guide will help me and others in the future._

## PRESET MODES

- FN + F4: Cycle through different RGB modes with all keys lite up.
- FN + F5: Cycle through different RGB modes with single keys lite up.

## USER MODES

- FN + F11: Cycle through animation using personalized colors (memory 1).
- FN + F12: Cycle through animation using personalized colors (memory 2).

## EDIT USER MODES

- FN + F9: Edit personalized colors in memory 1 (each individual key can have a different color).
- FN + F10: Edit personalized colors in memory 2 (each row can have a different color).
- FN + END: Exit edit mode.

_Notes: While in edit mode, use FN + ESC for color pallet or FN + F1/F2/F3._
_To set the individual color of the FN key, press "FN" + "Right CTRL" + "Any key you want to copy the color from". It doesn't work if you are in edit mode, you must be in normal typing mode._

## OTHER SHORTCUTS

- FN + UP/DOWN: Increase / decrease luminosity.
- FN + LEFT/RIGHT: Increase / decrease animation cycle speed.
- FN + R (for 5 seconds): Reset keyboard to factory settings.
- FN + 0: Turn all lights off.

## CHANGE KEY POSITIONS

- FN + F6: Disable Win key.
- FN + F7: Swap capslock and left Ctrl key.
- FN + F8: Swap Alt and Win key.
- FN + INS: Switch to Qwerty layout.
- FN + HOME: Switch to Dvorak layout.
- FN + PGUP: Switch to Colemark layout.

_Note: LED under right side of space bar will change color if any key swaps are active. Refer to manual._

## COUNTDOWN TIMER

- FN + ScrlLk to enter countdown timer setting mode.

_In countdown timer mode, F1-F12 represent 10-120 minutes, and 1-9 represent 1-9 minutes._
_For example, press F6 and 8 buttons to set a 68 minutes timer. Then press Enter to validate or End to exit._
